## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| harness shell → `opencode run` subprocess | The harness drives the CLI; CAIRN_LLM_* exported here must reach the plugins' direct `node <server>` shell-outs (not only mcp.environment) |
| OpenCode → `cairn-memory` stdio MCP subprocess | cairn-memory_memory_* tool calls cross here; server name/path must be exact |
| harness → filesystem config write | The scratch opencode.json MUST land under $SCRATCH_HOME, never the operator's real ~/.config/opencode |
| local model endpoint (loopback) → harness | Model output is only asserted against (canary FOUND/NOT-FOUND), never executed |
