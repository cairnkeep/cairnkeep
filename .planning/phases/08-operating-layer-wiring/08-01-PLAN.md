| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |

| T-08-SC | Tampering | npm/pip/cargo installs | low | accept | This phase installs zero external packages (RESEARCH.md Package Legitimacy Audit: N/A) — no supply-chain surface. |
