# Support

## Questions and usage help

Use [GitHub Discussions](https://github.com/cairnkeep/cairnkeep/discussions) for
installation questions, configuration help, and design discussion. Search the
documentation and existing discussions first. Include the Cairnkeep version,
operating system, Node.js version, client, and redacted `cairn doctor` output.

## Bugs and feature requests

Use the repository issue templates. Provide a minimal reproduction with
synthetic data. Do not attach memory databases, credentials, private endpoints,
or proprietary source material.

## Security

Do not report vulnerabilities in issues or discussions. Follow the private
reporting process in [SECURITY.md](SECURITY.md).

Support is community-based and provided on a best-effort basis. There is no
service-level agreement.
